# Employee Payroll System (EPS)

## 📌 Project Overview
The **Employee Payroll System (EPS)** is a server-side web application built using **Node.js**, **Express**, and **EJS**.  

The application allows users to:
- Manage employee records
- Calculate monthly payroll
- Perform full CRUD operations
- Persist data using a JSON file

---

## 🎯 Objective
Build a payroll management system that:
- Displays employee data in a dashboard
- Calculates tax and net salary dynamically
- Allows employee registration, editing, and deletion
- Stores all data inside `employees.json`
- Uses a custom module for file operations

---

## 🛠️ Tech Stack
- **Node.js**
- **Express.js**
- **EJS (Embedded JavaScript Templates)**
- **CSS**
- **File System (fs.promises)**
